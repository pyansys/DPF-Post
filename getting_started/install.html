Installation¶

Once you’ve installed Ansys 2021R1 or newer, you can install DPF with:

pip install ansys-dpf-post

This will install the latest version of ansys-dpf-post and all the necessary dependacies.

If you are unable to install the module on the host machine due to network isolation, download the latest release wheel at DPF-Post GitHub or from PyPi at DPF-Post PyPi

Editable Install (Development Mode)¶

If you wish to edit and potentially contribute to the DPF-Post python module, clone the repository and install it using pip with the -e development flag.

git clone https://github.com/pyansys/DPF-Post
cd DPF-Post
pip install -e .
